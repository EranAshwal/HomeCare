# 🏠 Home Care

A private household cleaner management app for tracking visits, payments, and messages.

**Live app:** open `index.html` in any browser, or visit the GitHub Pages URL for this repository.

## What it does
- 📅 Tracks Monday & Thursday cleaning visits on a calendar
- 💰 Manages hourly pay, commute reimbursement, and payment history
- 💬 Built-in chat between admin (Eran & Yahalom) and cleaner (Venus)
- 📷 Receipt and photo uploads
- 🇨🇦🇵🇭 Full bilingual interface (English / Filipino)
- 🔒 PIN-protected access with 15-minute idle auto-lock

## Tech
- Single HTML file — no build step
- React 18 + Babel (via CDN)
- Firebase Realtime Database for cross-device sync
- Hosted free on GitHub Pages

## Updating
Edit `index.html` directly on GitHub (pencil icon), commit, and the live URL refreshes within a minute.
